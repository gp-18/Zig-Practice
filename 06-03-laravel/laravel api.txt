what is an api ?
application programming interface ek set of rules aur protocols hai jo ek software application dusri application ke saath communicate karne ke liye use hota h . api ek bridge ki thara kaam karta hai jo do alg applications ke beech mein data aur functionality exchange karne mein help karta h . frontend h aur backend h vo dono ko ek saath communicate karne ka h to vo dono communicate kasie kaaregye api ki help se karegye na 

why do we use api ?
data sharing 
third-party integration 
modularity : code maintainability aur scalable karne ke liye 
platform independence : ek platform par develop ki gayi application ko doosre platform par bhi use kar skte h 

types of api 

restful api : representational state transfer  ek architecture style hai jo http protocol par based h simple aur scalable hota h  restful api http verb ( GET , POST , PUT , DELETE ) ka use karte h aur resource ko url ke zariye represent karta h . 

GraphQL Api : ek query language hai jo client ke requirements ke hisab se data retrive karne ke liye use hota h . client khud decide karta h usko kounsa data chhaiye 

SOAP api : simple object access protocol , ek protocol hai jo xml format mein data exchange karta h used for strict and secuirty feature
